# Gym Membership Management API

A simple RESTful API project for a gym membership management, exercising the concept of ORM (Object-Relational Mapping) using Spring Data JPA. 

Developed during a bootcamp from [Digital Innovation One](https://dio.me), based on the repository [Academia Digital](https://github.com/cami-la/academia-digital).
